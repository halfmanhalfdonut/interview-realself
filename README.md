# Pre-reqs
- [NodeJS](https://nodejs.org)

# Installing
- Clone this repo
- `npm install`

# Running the code
- `npm start ["line"] [length]`
- Alternatively, `node index.js ["line"] [length]`
- E.g. `npm start "The quick brown fox jumps over the lazy dog." 52`

# Testing the code
- `npm test`
